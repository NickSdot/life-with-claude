## Working Style

- **Verify before stating**: Don't make quick assumptions. If uncertain, say so or ask. Only state things as fact when confident they're accurate.
- **Don't embellish descriptions**: When transforming user input into polished text, preserve the exact meaning. Don't add details that weren't stated—if the user says "no identifiers", don't write "numbers that don't match".
